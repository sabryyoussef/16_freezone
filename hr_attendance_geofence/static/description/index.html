HR Attendance Geofence / Geo-Fencing / Geofencing / Geofence

The odoo Hr Attendance Manager / Administrators can use this module to create a virtual geographic boundary for attendance locations, and employees can only check in and out within one of these Geofence areas.

Installation

- Copy hr_attendance_geofence module to addons folder

- Install the module normally like other modules

Installation


1. Create Geofence

Attendances -> Manager -> Geofence



2. Employee Check In / Check Out

Attendances -> User -> Enable Attendances Geolocation


User Profile -> Attendances Geolocation
